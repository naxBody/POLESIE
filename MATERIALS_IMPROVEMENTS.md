# Улучшения страницы "Материалы"

## Изменения

### 1. Красивое отображение количества товаров

Добавлено наглядное отображение остатков на складе с использованием:
- **Цветовой индикации** (зелёный/жёлтый/красный)
- **Прогресс-бара** с анимацией shimmer-эффекта
- **Крупного числового значения** с единицами измерения

#### Уровни запасов:
- 🟢 **Высокий** (>50 ед.) - зелёный цвет
- 🟡 **Средний** (10-50 ед.) - жёлтый цвет  
- 🔴 **Низкий** (<10 ед.) - красный цвет

#### Отображение в двух видах:
- **Карточки**: Полная версия с подписью "📦 На складе", значением и прогресс-баром
- **Таблица**: Компактная версия со значением и мини-прогресс-баром

### 2. Динамический поиск

Улучшен существующий поиск с добавлением:
- **Автоматической отправки** через 500мс после прекращения ввода (debounce)
- **Расширенного placeholder** с подсказкой
- **Визуального выделения** поля поиска (на всю ширину)
- **Подсказки** о возможностях поиска

#### Поиск работает по:
- Названию материала (полному и краткому)
- Коду материала (внутреннему артикулу)
- Названию категории
- Названию подкатегории
- ГОСТ/стандарту

## Технические детали

### CSS стили
Добавлены новые классы для отображения количества:
- `.quantity-wrapper` - обёртка блока количества
- `.quantity-label` - подпись "На складе"
- `.quantity-display` - контейнер для значения и прогресс-бара
- `.quantity-value-*` - стили для числового значения (high/medium/low/none)
- `.quantity-bar-container` - контейнер прогресс-бара
- `.quantity-bar-*` - стили прогресс-бара с градиентами
- `.quantity-badge-*` - обновлённые стили бейджей

### JavaScript функции
- `debouncedSearch()` - функция задержки перед отправкой формы поиска

### PHP логика
Обновлена логика определения стиля отображения количества как в карточках, так и в таблице.
