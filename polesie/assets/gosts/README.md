# Папка для документов ГОСТ

## Как добавить документы ГОСТ

### Вариант 1: Скачать с официальных источников

1. **Росстандарт (official)** - https://rskg.ru/
2. **Техэксперт** - https://techexpert.pro/
3. **КонсультантПлюс** - https://www.consultant.ru/
4. **Гарант** - https://www.garant.ru/

### Вариант 2: Использовать онлайн-ресурсы

Сейчас система использует ссылки на внешний ресурс docs.cntd.ru для открытия ГОСТов.

### Формат именования файлов

Файлы должны именоваться по следующему формату:
```
GOST-{номер}-{год}.pdf
```

Примеры:
- `GOST-1050-2013.pdf`
- `GOST-4543-2016.pdf`
- `GOST-14637-2016.pdf`

## Интеграция локальных файлов

Для использования локальных файлов вместо внешних ссылок, измените функцию `generateGostLink()` в файле `modules/warehouse/materials.php`:

```javascript
function generateGostLink(gostString) {
    const gostNumber = extractGostNumber(gostString);
    if (!gostNumber) {
        return '#';
    }
    
    // Для локальных файлов
    const yearMatch = gostString.match(/-(\d{4})$/);
    const year = yearMatch ? yearMatch[1] : '';
    const fileName = `GOST-${gostNumber.replace('.', '-')}${year ? '-' + year : ''}.pdf`;
    
    return `../../assets/gosts/${fileName}`;
}
```
